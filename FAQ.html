Nobody has actually asked any questions yet, but I thought I'd put some stuff here that somebody would probably ask questions about

  • Q: What's the difference between a full build and source code?
    A: A full build is all the files produced when I built this project. It's only really useful if the one file doesn't work.
    Source code means all the code that was used to make this project. If you want to mod or port this app, this is what you need.

  • Q: Why isn't there a full build option in versions 1.0.2 and above?
    A: The full build file is included in the source code. It's the "dist" and "build" folders.

  • Q: What's the latest version?
    A: Version 1.0.2 for windows, no versions are out for mac or linux

  • Q: Does this program support Mac and Linux?
    A: Not currently. There are plans for porting to Mac, however this will be updated slower. There are no plans to port to Linux.

  • Q: Can I add my own options to the generator?
    A: Not without downloading and modifiying the source code. If you want to add anything that isn't a feature, simply add more entries to the json file.
    If you want to add a feature without it doing anything, do as above, but don't add it to the end of the list. "Multi Feature" must be the last element in the list for it to work correctly.
    If you want to add a new category, or make a feature that does something, it requires knowledge of python. I might make a tutorial on how to do it someday.

  • Q: I've modified source code, and want to share the result with my friends. How do I do that?
    A: Install pyinstaller (Look it up), then run
    'pyinstaller --onefile --windowed --add-data "npcs.json;." --add-data "NPC Data;NPC Data" main.py' without the quotation marks at the start and end.
    That will give you a "build" and "dist" folder. The .exe is found in the dist folder.
